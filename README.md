🌊 Sistema de Monitoramento de Reservatório de Água

📖 Sobre o Projeto<br>
<br>Este projeto foi desenvolvido em Python com o objetivo de simular um sistema simples de monitoramento do nível de água de um reservatório.

<br>A aplicação utiliza a biblioteca Colorama para exibir mensagens coloridas no terminal, permitindo uma visualização intuitiva da situação do reservatório conforme o nível informado pelo usuário.

✨ Funcionalidades<br><br>
✅ Monitoramento de cinco níveis do reservatório;<br>
🎨 Exibição de mensagens coloridas;<br>
⚠️ Alertas visuais conforme o nível da água;<br>
🛡️ Tratamento de entradas inválidas;<br>
💻 Interface simples via terminal.<br><br>
🛠️ Tecnologias Utilizadas<br><br>
🐍Python<br>
🎨Colorama<br><br>
📊 Níveis do Reservatório<br><br>
Nível	Situação	Cor<br>
1	Muito baixo (crítico)	🔴 Vermelho<br>
2	Baixo	🟡 Amarelo<br>
3	Médio	🟢 Verde<br>
4	Alto	🔵 Ciano<br>
5	Muito alto (alerta)	🔷 Azul<br>
